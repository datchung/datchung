True Txt Reader Writer

What is it?
Read and write text files (.txt). Since the files are saved in the .txt format, they can be easily shared across devices. Use this app for quick notes, reminders, or even lengthy text. Now with the capability to read large text files.
Native app that works on Android 2.1 and up. Extra attention has been paid to support older versions of Android.
Why?
While there are other notepad/editor apps available, many of them save files to the cloud or in their own format. This one doesn't. Saving to the cloud is not ideal if the user has a limited network. Saving in their own format may not be open and may require special software which limits sharing across devices.

Learn [English / French / Italian / Portuguese / Vietnamese / Spanish]

What is it?
A quiz-based game that aims to improve the user's vocabulary in another language. A word/phrase is shown in the first language and multiple choice options are shown in the second language. After the user answers, feedback will be given depending on correct/incorrect choice.
Hybrid app that works on Android 2.2 and up. Extra attention has been paid to support older versions of Android and multiple platforms (web and Windows Phone). Uses a PHP web service and the Google Translation API.
Why?
Gamification of learning a language, making it fun.
