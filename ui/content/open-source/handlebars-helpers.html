What is it?

A collection of helpers for Handlebars.js, a templating engine that is compatible with mustache.

The demo page contains instructions for getting started and examples for every helper included in this library.

Why?

Handlebars.js is shipped with a couple of built-in helpers. The additional helpers provided in this library enable easier and faster coding and continue with the idea of semantic templates.
